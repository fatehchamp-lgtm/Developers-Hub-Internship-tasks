# Task 6: Real Estate Valuation Engine

## Project Objective
An enterprise-grade predictive analytics implementation designed to estimate residential property valuations using advanced regression paradigms. The core architecture evaluates multiple multi-dimensional property features to output accurate structural pricing benchmarks.

## Technical Framework
- **Data Engineering:** Automated categorical transformations via scikit-learn pipeline nodes.
- **Algorithms Implemented:** Baseline Ordinary Least Squares (Linear Regression) alongside Gradient Boosting Decision Trees.
- **Performance Evaluation:** Validated using Mean Absolute Error (MAE), Root Mean Squared Error (RMSE), and R² coefficient metrics.

## Key Operational Insights
- Iterative ensemble boosting strategies successfully capture non-linear market interactions compared to simple linear configurations.
- Spatial bounding metrics (`Area` and `Location`) serve as the primary drivers of localized real estate evaluation shifts.
